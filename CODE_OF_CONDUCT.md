# Code of Conduct

CellKit is an open-source developer tool project. The project should remain welcoming, practical, and focused on helping developers build safely on CKB.

## Expected Behavior

Participants are expected to:

- Be respectful and constructive
- Assume good intent where reasonable
- Keep discussions focused on the project and technical issues
- Give clear feedback without personal attacks
- Respect different experience levels
- Avoid sharing private information without permission

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or intimidation
- Personal attacks
- Discriminatory language
- Publishing private information
- Spam or repeated off-topic disruption
- Encouraging unsafe handling of private keys, seed phrases, or funds

## Project Safety Norms

CellKit is private-key-free and testnet-first. Contributors should avoid suggesting or adding flows that request private keys, seed phrases, wallet custody, hidden signing, or unclear broadcast behavior.
